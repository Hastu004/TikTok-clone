## About The Project

TikTok clone is a educational proyect which tries to reproduce the famous TikTok app with React and Vite

## Getting Started

Install the dependencies:

```sh
$ npm install
// or in mac OS
$ yarn
```

Run in dev mode:

```sh
$ npm run dev
// or in mac OS
$ yarn dev 
```
